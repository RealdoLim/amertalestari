# CV. Amerta Lestari Website

This project is a company profile website for **CV. Amerta Lestari**, a business engaged in the distribution and sale of fertilizer products.  
The website is designed to present the company professionally, showcase its main products, and make it easy for customers or business partners to get in touch.

## Overview

The website was built as a clean and responsive company profile site using only:

- **HTML**
- **CSS**
- **JavaScript**

It is intended to be lightweight, easy to maintain, and suitable for deployment as a static website.

The main purpose of this website is to:

- introduce **CV. Amerta Lestari**
- display the company’s fertilizer products
- provide clear contact information
- support trust and credibility for customers and business partners

---

## Main Features

- Modern responsive landing page
- Company profile / About Us section
- Product showcase section
- Product cards with images
- Contact information section
- WhatsApp contact button
- Email information
- Mobile-friendly layout
- Clean and professional visual design

---

## Website Sections

### Home
Introduces the business and presents the main value of the company.

### About Us
Explains the company as a distributor and seller of fertilizer products, with a professional business profile description.

### Products
Displays the main products offered by the company:

Each product is displayed with:
- product name
- short description
- image card

### Contact
Provides direct contact details for customers or partners, including:
- phone / WhatsApp number
- email
- other business contact details if needed

---

## Tech Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**

No framework is used in this project.
